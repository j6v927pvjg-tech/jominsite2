# 城民 公式サイト

いだちゃんねるファンコミュニティアプリ「城民」の公式サイトです。

## ファイル構成

```
├── index.html          トップページ
├── terms.html          利用規約
├── privacy.html        プライバシーポリシー
├── guideline.html      コミュニティガイドライン
├── unofficial.html     非公式アプリについて
├── contact.html        お問い合わせ
├── style.css           スタイルシート
├── script.js           JavaScript
└── images/             画像フォルダ
```

## GitHub Pages での公開方法

1. このリポジトリを `Settings` → `Pages` から公開設定する
2. Branch: `main` / フォルダ: `/ (root)` を選択して Save
